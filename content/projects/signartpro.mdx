## Overview

CredLens is an augmented-reality (AR) fintech concept I designed during college as part of a personal exploration into futuristic interface systems. The goal was to rethink how users interact with personal-finance data by transforming a simple, everyday object—a debit/credit card—into a gateway for dynamic financial insights.

The project uses spatial UI principles, transparent glass-like components, and intuitive data visualization to demonstrate how AR can make financial information more accessible, contextual, and engaging.

## The Problem

Traditional banking apps confine users to small screens, nested menus, and static charts.

Common user pain points include:

- **Limited visibility into overall financial health**
- **Fragmented data** (balance, transactions, categories all separated)
- **Cognitive overload** from text-heavy interfaces
- **Lack of emotional connection** or immediacy
- **Difficulty understanding spending patterns** over time

With AR becoming increasingly mainstream, I wanted to explore:

> What if personal finance wasn't hidden behind an app—but was instantly visible in augmented reality?

## The Vision

CredLens imagines a future where financial data becomes ambient, spatial, and triggered by context.

Holding or viewing your physical credit card becomes an AR portal that reveals:

- Your current balance
- Monthly income & expenses
- Recent transactions
- Category-wise spending
- A visual summary of financial habits

This transforms a passive object (the card) into an active financial lens—hence the name CredLens.

## Core Concept

**Object-Anchored AR UI**

The physical card acts as a tracked object. Once recognized, holographic UI elements appear locked around it.

**Spatial Financial Dashboard**

Users see a complete, glanceable overview of their finances floating in their environment.

**Minimal, Glass-Layered UI System**

Panels are semi-transparent, light, and clean—reducing clutter and enhancing legibility in AR.

## Key Features Designed

### (1) Balance & Monthly Summary Panel

A floating display showing:

- Current balance
- Total income for the month
- Total expenses

Designed for quick mental processing with clear color cues.

### (2) AR Transaction Timeline

Horizontal scrollable list with:

- Icons for categories
- Amounts (credit/debit)
- Time/date indicators

The goal was to create a timeline that feels natural in 3D space.

### (3) Spending by Category Visualization

A set of color-coded progress bars showing:

- Shopping
- Food
- Transport

Each category is mapped to red/yellow/green to quickly communicate intensity.

### (4) Icon-Based Navigation Menu

Floating navigation icons allow users to switch between:

- Transactions
- Analytics
- Spending categories
- User profile

This demonstrates how app navigation works when freed from a flat screen.

## Interaction Flow

1. User holds their card in front of the camera
2. CredLens recognizes the card's shape/pattern
3. UI panels anchor themselves around the card
4. Financial insights appear in a 3D layout
5. User can move the card, and UI moves with it
6. Panels rearrange based on orientation for optimal readability

The interaction is designed to feel magical yet grounded.

## Design Decisions

### Spatial Usability

AR interfaces can easily become cluttered. I focused on maintaining:

- Minimal text
- Bold numeric values
- High-contrast foreground elements
- Subtle motion to indicate depth

### Color Strategy

- **Red** → High spending
- **Yellow** → Medium spending
- **Green** → Low or healthy spending
- **White/Neutrals** → Labels and universal information

This keeps the interface readable in various lighting environments.

### Typography

Clean, geometric typefaces were used to enhance futuristic clarity.

### Glassmorphic Component Library

To create depth without overwhelming the user, every panel uses:

- Semi-transparent backgrounds
- Soft shadows
- Rounded corners
- Layered hierarchy

## Tools & Methods

- **Figma** for UI and visual exploration
- **After Effects / Premiere** for motion and demo assembly
- **AR anchoring prototypes** (using simulated object tracking)
- **Video overlay techniques** to visualize the final interaction

This was a conceptual project, not a functional AR app.

## Final Outcome

The final result is an immersive AR financial dashboard that:

- Turns a simple card into an information hub
- Makes personal finance feel intuitive and visual
- Breaks away from traditional 2D mobile banking patterns
- Demonstrates my early interest in spatial design, interaction design, and AR usability

The prototype video showcases how a user would explore their finances effortlessly through a futuristic augmented interface.

## What I Learned

- Designing for AR requires a different mental framework than for mobile
- Depth, motion, and spatial placement are as important as typography
- Visual noise becomes a major challenge in 3D environments
- Anchoring UI to physical objects creates powerful contextual interactions
- Simplicity and hierarchy are crucial in mixed reality

This project helped me grow significantly in conceptual thinking and UI innovation.

## Future Improvements

If expanded, CredLens could include:

- Budget forecasting using AI
- Gesture-based interactions
- Cross-device syncing
- Real AR object detection
- Personalized financial coaching

It could extend into a comprehensive AR personal-finance ecosystem.
