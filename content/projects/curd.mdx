## Overview

Curd is a social media platform where I led UX research and designed high-fidelity prototypes to enhance user engagement and usability. The project involved comprehensive user journey mapping, iterative design improvements, and close collaboration with developers to bring the vision to life.

## The Problem

The platform needed to improve user engagement and create a more intuitive social media experience. Users were struggling with navigation, content discovery, and overall platform usability.

## The Solution

Through extensive UX research and iterative design, I created user journey maps and high-fidelity prototypes that addressed key pain points. The redesigned experience focuses on intuitive navigation, improved content discovery, and seamless user interactions.

### Key Features

**User Journey Mapping**
Comprehensive mapping of user flows from onboarding to daily engagement, identifying friction points and opportunities for improvement.

![User journey mapping and iPhone mockups showing the design process](/images/projects/curd/iphone-mockup-front-back.png)

**High-Fidelity Prototypes**
Interactive prototypes that demonstrate the redesigned experience, allowing for stakeholder feedback and developer handoff.

**Usability Enhancements**
Improved navigation patterns, content discovery mechanisms, and engagement features that make the platform more intuitive and enjoyable to use.

![Clay iPhone mockup showcasing the redesigned interface](/images/projects/curd/clay-iphone-mockup.png)

## Impact

The redesigned experience improved user engagement metrics and created a more cohesive user journey. Collaboration with developers ensured that design decisions were implemented effectively, resulting in a more polished and user-friendly platform.

## My Role

I owned the UX research, user journey mapping, and high-fidelity prototype design. I collaborated closely with developers to ensure design feasibility and implementation quality, creating a seamless bridge between research insights and final product.
